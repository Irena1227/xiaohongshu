# MildGarden · 封面生成器

> 一个人和她的 AI 一起造的小红书封面工具。
> 纯 HTML 单文件，没有后端，没有框架，打开就用。

---

## 它是什么

一个小红书封面图生成器。选模板 → 改文字 → 导出 1080×1440 高清图。

为 [MildGarden](https://www.xiaohongshu.com/user/profile/xxx) 小红书账号定制设计，但所有文字都可以自定义，谁都能用。

## 功能

- **实时预览** — 改字即刻看到效果
- **字号滑块** — 70%~140% 无级调节
- **字间距滑块** — -2px ~ +12px 微调
- **文字强调** — 指定关键词变红色（日常碎片模板）
- **图片上传** — 语言之外模板支持上传图片做背景
- **自适应字号** — 文字多了自动缩小，不会溢出
- **粘贴去格式** — 从任何地方粘贴文字，自动剥离富文本
- **导出 1080×1440** — 小红书封面标准比例，高清 PNG
- **手机适配** — 手机浏览器打开可用，长按保存或点击下载
- **桌面端** — 点导出直接下载
- **PWA 支持** — 添加到手机主屏幕当伪 APP 用
- **顶部 CONFIG** — 打开源码改默认字号/间距/行高，刷新生效

## 技术栈

纯前端，零依赖安装：

- **单 HTML 文件** — 全部代码在一个文件里
- **html2canvas** — 大部分模板的导出渲染（CDN 加载）
- **Canvas 原生绘制** — 语言之外模板单独用 Canvas API 导出，绕开 html2canvas 的图片兼容问题
- **Google Fonts** — Cormorant Garamond + Noto Serif SC + DM Sans
- **像素风 SVG 装饰** — 玫瑰、星星、猫、爱心、莫比乌斯环、手柄，全部手写 SVG

## 使用方式

### 方式一：GitHub Pages（推荐）

Fork 本仓库 → Settings → Pages → 选 main 分支 → 拿到你的链接，手机浏览器打开即用。

### 方式二：本地使用

下载 `index.html`，双击打开，完事。不需要装任何东西。

## 自定义

打开 `index.html`，最上面有一个 `CONFIG` 区块：

```javascript
const CONFIG = {
  defaultFontPercent: 108,        // 字号滑块初始值
  defaultExtraLetterSpacing: 0,   // 额外字间距(px)
  defaultLineHeight: 1.9,         // 行间距倍数
};
```

各模板的默认文字在 `TC` 对象的 `defaults` 里改。
